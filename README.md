# How to run the tests
1. Clone the repository:
```bash
 git clone https://github.com/vyavna/web-aqa.git
```
2. Run the app
```bash
npm run dev
```
3. Install playwright:
```bash
npm init playwright@latest --legacy-peer-deps
```
4. Run all tests
```bash
 npx playwright test  
 ``` 
 5. See the report
```bash
npx playwright show-report
